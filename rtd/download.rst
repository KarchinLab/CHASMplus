CHASMplus releases
------------------

* `CHASMplus v1.0.0 <https://github.com/KarchinLab/CHASMplus/archive/v1.0.0.tar.gz>`_ - 8/17/2018 - Initial release


Necessary additional code
-------------------------

* `20/20+ code <https://github.com/KarchinLab/2020plus/releases>`_ produces driver gene scores. Please follow installation instructions from the `20/20+ website <http://2020plus.readthedocs.io/>`_.
* `SNVBox code <https://www.dropbox.com/scl/fi/tkaysovawrl6euhc033id/SNVBox.tar.gz?rlkey=oz30lj02xbsjcmkugb7nrfoh4&st=l52p3swv&dl=1>`_ fetches the features used by CHASMplus from a MySQL database
